# ClassForum – UCC Dominase Campus

Closed student forum for **AH/MDL/25** at the University of Cape Coast, Dominase Campus.

## Tech Stack
- React 18 + Vite
- No backend — data stored in localStorage
- Deployed on Vercel

## Local Development
```bash
npm install
npm run dev
```

## Deploy
Push to GitHub → Vercel auto-deploys on every commit.

## Admin Access
- Superadmin: AH/MDL/25/0002
- Course Reps (Admin): AH/MDL/25/0067, AH/MDL/25/0050
